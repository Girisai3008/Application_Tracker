☁️ GitHub Gist Auto-Sync

Enter your GitHub Personal Access Token once — your data will auto-save to a private secret Gist every time you make a change. Open the app on any device with the same token to instantly load all your data.

Create at: github.com/settings/tokens/new → Name it JobTrack Sync → Expiration: No expiration → Check only gist scope → Generate → Copy & paste above.
If you already set up sync on another device, copy the Gist ID from there (shown in the green box below after connecting) and paste it here before clicking Connect.

📋 Step-by-Step: Create a Token

1. Go to github.com/settings/tokens/new
2. Note (name): JobTrack Sync
3. Expiration: No expiration (or 1 year)
4. Scopes: check only ✅ gist
5. Click Generate token
6. Copy the token (starts with ghp_) and paste it above
7. Click Connect — done! All your data syncs automatically from now on.

🔒 Privacy & Security

Your token is stored only in this browser's localStorage — never sent anywhere except directly to the GitHub API (api.github.com). The Gist is secret (private, unlisted). Only someone with your token can read it.
